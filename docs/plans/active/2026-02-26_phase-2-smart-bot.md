## Контекст

Вторая фаза развития TG Planer (см. [docs/DOMAIN.md](../../DOMAIN.md)). Бот становится умнее: понимает голосовые сообщения, формирует план дня, управляет просроченными задачами и отправляет контекстные уведомления с inline-кнопками.

### Вопросы и ответы

**Q:** Какой сервис для распознавания речи?
**A:** OpenAI Whisper API (~$0.006/мин) — лучшее соотношение цена/качество для русского языка.

**Q:** Как реализовать повторяющиеся напоминания?
**A:** Библиотека `rrule` (RFC 5545) — стандарт для описания повторений.

**Q:** Как учитывать timezone пользователя для дайджестов?
**A:** Cron каждые 15 минут проверяет `users.digest_time` с учётом timezone.

## Решение

Расширение бота из Phase 1: интеграция Whisper API для голоса, cron-based дайджесты (утро/вечер), автоформирование "Моего дня" на основе дедлайнов и приоритетов, библиотека rrule для повторяющихся напоминаний. Все уведомления получают inline-кнопки для быстрых действий.

### Альтернативы

1. **Google Speech-to-Text вместо Whisper** — дороже, сложнее настройка, сопоставимое качество
2. **Push-уведомления вместо cron** — Telegram не поддерживает push, только отправка сообщений
3. **Хранение score приоритета в БД** — избыточно, пересчёт при запросе проще и актуальнее

## Последствия

### Позитивные
- Голосовой ввод — главное преимущество перед текстовыми планировщиками
- Автоматический "Мой день" решает проблему "не знаю за что браться"
- Дайджесты формируют привычку пользоваться ботом

### Риски
- Whisper API latency (2-5 сек на распознавание) — нужен typing indicator
- Cron каждые 15 минут — дайджест может опоздать до 15 мин от настроенного времени
- Сложность алгоритма автоприоритета — нужно тестировать на реальных данных

### Технический долг
- Нет тестов для алгоритма скоринга
- Hardcoded веса в алгоритме автоприоритета (вынести в конфиг позже)

## Тестирование

- Голосовое сообщение с одной задачей — корректная транскрипция и парсинг
- Голосовое с несколькими задачами — все задачи распознаны и показаны
- Утренний дайджест приходит в настроенное время (±15 мин)
- Просроченная задача появляется в дайджесте с кнопками
- Повторяющееся напоминание срабатывает и создаёт следующее
- Inline-кнопки на уведомлениях обновляют задачу и сообщение
